## 功能要求

- 有登陆界面，只要输入一个昵称，且要检查昵称是否已经被其它用户使用
- 游戏页面，一个猜数字的游戏，随机生成一个0-50之间的整数，要求玩家输入自己猜的，并对用户输入数字进行检查，并对输入情况进行反馈，连续6次没有猜对则游戏失败，游戏成功猜对要将情况记入排行榜，询问用户是否开始新的游戏。
- 排行榜页面，显示所有参加游戏的用户的游戏情况
